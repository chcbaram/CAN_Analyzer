# CAN_Analyzer
---
## 개요 
CAN Analyzer는 네이버 !전자공작 카페(https://cafe.naver.com/circuitsmanual) 에서 진행 중인 CAN통신 분석 모듈 제작 프로젝트입니다.

<br>

F/W 개발환경
-
+ STM32CubeIDE 1.7.0
+ STM32CubeMX 6.3.0

<br>

CAN Analyzer 보드 
-
STM32H723VG를 사용하고 상용 케이스에 맞게 PCB를 제작 중입니다.
